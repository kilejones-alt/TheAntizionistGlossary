EASY WINDOWS 7 UPLOAD VERSION

This package has NO folders.

Upload every file in this folder to GitHub at the top level.

Steps:
1. Unzip this zip.
2. Open the unzipped folder.
3. In GitHub, click Add file > Upload files.
4. Select all files in this folder.
5. Click Open.
6. Click Commit changes.
7. Wait 1-3 minutes.
8. Refresh your site with Ctrl + F5.

Do not worry about assets/ or entries/ folders in this version. They were flattened so old Windows upload works.
