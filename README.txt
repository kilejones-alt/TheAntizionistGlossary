The Antizionist Glossary Website — Version 1

Open index.html in a browser to preview.

To replace the author photo:
1. Put your photo in assets/author.jpg or assets/author.png.
2. In index.html, replace assets/author-placeholder.svg with your photo filename.

To edit entries:
Open any file in entries/ and paste the final entry text into the existing sections.

Free hosting:
Netlify: drag this whole folder into a new Netlify site.
GitHub Pages: upload the folder contents to a repository and enable Pages.

This is a static HTML/CSS/JS site. No database is required.
