UPLOAD INSTRUCTIONS

1. Download this zip.
2. If Windows shows Extract all files, click it.
3. If extraction is confusing: open the zip, press Ctrl + A, press Ctrl + C, create a new folder on Desktop called upload, open it, press Ctrl + V.
4. Open the extracted/copied folder.
5. Press Ctrl + A.
6. GitHub → Add file → Upload files.
7. Choose all files.
8. Click Commit changes.
9. Wait 1–3 minutes.
10. Refresh the site with Ctrl + F5.
